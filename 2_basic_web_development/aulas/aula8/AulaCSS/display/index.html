Aula Display em CSS

O atributo display em CSS define como um elemento HTML deve ser exibido no layout da página.

display: block;

Elementos com display: block; são renderizados como blocos que ocupam toda a largura disponível e começam em uma nova linha. Eles geralmente são usados para criar divisões entre conteúdo.

    Exemplos:

  • Parágrafos: <p> Este é um exemplo de parágrafo.</p>
  • Títulos: <h1>Este é um exemplo de título.</h1>
  • Divisões: <div>Este é um exemplo de divisão.</div>
  • Listas: <ul><li> Este é um exemplo de lista.</li><ul>

display: inline;
display: inline;
display: inline;
display: inline;

Elementos com display: inline; são renderizados em linha e ocupam apenas o espaço necessário.

    Exemplos:

  • Links: <a href="#">Este é um exemplo de link</a>
  • Texto: <span>Este é um exemplo de texto em linha</span>.
  • Ícones: <i class="fas fa-star"></i> Este é um exemplo de ícone em linha.
  • Botões: <button>Este é um exemplo de botão</button>

display: inline-block; display: inline-block; display: inline-block; display: inline-block;

Elementos com display: inline-block; também são renderizados em linha, mas permitem definir largura e altura, comportando-se como blocos em linha.

    Exemplo:

  • img: <img scr="">Este é um exemplo de imagem</img>

display: none;

Elementos com display: none; não são renderizados na página e ficam invisíveis.Eles são usados para ocultar temporariamente elementos via CSS.
